구글 앱시트 Google AppSheet는 엑셀이나 구글 스프레드시트 등을 이용하여 코딩 없이 손쉽게 데이터 기반의 모바일앱/모바일웹을 만들 수 있는 도구입니다. ▷ 노코드 앱제작 툴 앱시트 간략 소개

이 페이지는 제가 앱시트를 직접 사용해본 후 국내 사용자들이 쉽게 배워보실 수 있도록 정리한 글모음입니다. 아래 각 폴더의 제목을 누르면 상세 페이지 링크를 볼 수 있습니다. 키워드로 관련 글을 찾고자 하는 경우 Ctrl + F를 눌러 검색해보시기 바랍니다.

  • 🟩 앱시트 기능별 앱 제작방법

    테이블 구조

    • [KEY] 행마다 고유한 번호(key) 부여하기 - UNIQUEID() 함수 활용
    • [KEY] 키(key) 설정에 대한 여러가지 고찰 - 무엇을 key로 선택할까? 연속적으로 증가하는 key는 어떻게?
    • [KEY] 테이블에서 Key가 될 수 있는 컬럼이 있는지 확인하고 없으면 만들어 쓰자
    • [Ref] 다른 데이터베이스(테이블)을 연결하고 참조하기
    • [Ref] 상하위 테이블 관계 설정 Is a part of?
    • [Ref] 자식테이블에 부모테이블 정보 자동 입력하기, 한 번에 여러 행 입력
    • [Ref] Ref 타입을 적극 활용하되 하나의 관계는 한번만 설정하자
    • [Slice] 특정 조건의 데이터만 보여주는 슬라이스(부분 테이블)
    • [Slice] 특정 조건의 마지막행만 가져오는 슬라이스 만들기 - MAXROW 함수
    • [Slice] 최근 한 달 데이터만 볼 수 있는 슬라이스 만들고 차트 뷰로 연결하기 📺
    • 구글 캘린더 google calendar 연동하기

    컬럼의 유형과 설정

    • [Enum] 미리 설정된 목록 중 선택할 수 있게 하는 컬럼 타입 Enum, Enumlist
    • [Enum] 상위 Enum 선택에 따라 하위 Enum 항목이 바뀌는 계층적 선택지 만들기
    • [Enum] Enumlist(다중 선택) 목록에서 다른 테이블의 값이 표시되게 하기
    • [Image] 이미지 삽입, 이미지 연동 방법 Image type
    • [Image] 구글 스프레드시트에서 앱시트 업로드 이미지를 바로 미리보기하는 방법
    • [Url] Url 타입 이용하여 지도앱, 네비앱 (네이버지도, 카카오맵, 티맵) 연결하기
    • [Url] 컬럼 값을 이용하여 검색사이트 연결하기 - ENCODEURL 함수
    • [Scannable] 바코드나 QR코드를 이용하여 제품정보 입력하고 검색하는 앱 만들기
    • 주소 자동 완성(Address type), GPS 현재 위치 입력(LatLong type)
    • 서명 넣기(Signature type)와 그림 그려서 넣기(Drawing type)
    • 컬럼 내용을 수정한 일시와 수정한 횟수를 캡쳐하기 - changetimestamp, changecounter
    • [가상컬럼] 연도별 월별 차트뷰 만들기
    • [가상컬럼] 날짜 컬럼에 요일도 같이 표시 하기
    • [가상컬럼] 데크 뷰에 여러 컬럼값 보이게 하기, 경과시간 계산하기
    • [Formula] 다른 컬럼에서 선택한 값에 따라 컬럼의 기본값을 다르게 셋팅하기 - IF, IFS 함수
    • [Formula] 드롭다운 항목을 선택하면 다른 테이블의 이미지를 참조하여 불러오기
    • [Formula] 특정 조건이 일치하는 다른 행의 값을 참조하는 방법 - SELECT 함수, FILTER 함수
    • [Formula] 로그인 계정을 인식하여 이름으로 바꾸기 - LOOKUP 함수 응용
    • 구글 스프레드시트 함수 연동하기 - Spreadsheet formula 📺
    • 구글 스프레드시트 함수 연동하기 - Spreadsheet formula 📝
    • 구글 시트 SUMIF 함수로 입출고정보를 제품정보 테이블에 적용

    화면 구성

    • [Group] 데이터를 계층적 그룹으로 묶어서 보여주기 - Group by
    • [Group] Group aggregate 활용하여 집계화면 만들기
    • 구글 스프레드시트의 QUERY 기능을 이용하여 집계화면 만들기
    • [Format rules] 조건에 따라 항목별 서식 적용하기
    • [Format rules] Group by에 의한 그룹명에 포맷 룰 서식 설정
    • [Dashboard] 한 페이지에 여러개의 화면을 탭으로 넣기 - Mobile Tab view
    • [Dashboard] 선택한 항목에 따라 달라지는 반응형 차트 만들기 - Interactive Dashboard
    • [Form] 폼 뷰에서 안내문구나 이미지 등 보여주기 목적의 컬럼 만들기 - Show type
    • [Form] 여러 페이지 여러 섹션으로 된 폼 뷰 만들기 - Show type
    • [Form] 데이터 추가/변경 상황에 따라 편집 가능 여부 다르게 설정하기 - Editable?
    • [Form] Enum 값에 따라 보였다 숨겼다하기, 값이 없어도 디테일 뷰에 표시되게하기 - Show? new
    • [Map] 주소 또는 위도 경도 데이터로 맵 뷰 만들기
    • [Map] XY type을 이용하여 나만의 Map view 만들기 📺
    • 동적으로 바뀌는 뷰 네임(View name)은 Display name에서
    • 버튼이름, 컬럼이름을 화면에 따라 다르게 표시하기, CONTEXT() 함수
    • 다른 테이블의 참조내역을 보여주는 인라인 뷰(inline view) 편집하기
    • Quick edit 기능으로 디테일 뷰에서 데이터 조회와 수정을 동시에
    • Enable QuickEdit 기능으로 목록 뷰에서 여러 컬럼을 동시에 수정하기
    • 앱 사용법 안내 페이지 만들기 - About 화면 활용
    • 갤러리 뷰를 이용해 각 화면으로 연결되는 메뉴 페이지 만들기 - LINKTOVIEW 함수
