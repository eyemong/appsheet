구글 앱시트 Google AppSheet는 엑셀이나 구글 스프레드시트 등을 이용하여 코딩 없이 손쉽게 데이터 기반의 모바일앱/모바일웹을 만들 수 있는 도구입니다. ▷ 노코드 앱제작 툴 앱시트 간략 소개

이 페이지는 제가 앱시트를 직접 사용해보고 사용법을 쉽게 알아볼 수 있도록 정리한 글모음입니다. 각 폴더의 제목을 누르면 상세 페이지 링크를 볼 수 있습니다.

  • 📁기능별 앱 제작방법

    컬럼의 유형

    • Enum, Enumlist: 미리 설정된 값을 선택할 수 있게
    • Image: 이미지 삽입, 이미지 연동하기
    • Changetimestamp, Changecounter: 컬럼 내용을 수정한 일시와 수정한 횟수를 캡쳐
    • Signature, Drawing: 서명 넣기와 그림 그리기
    • Show: 폼 뷰에서 이미지나 텍스트를 보여주는 컬럼
    • Url: 컬럼 내용을 이용하여 링크 주소 연결하기
    • Url: 지도앱, 네비앱(네이버지도, 카카오맵, 티맵) 연결하기
    • 상위 enum 선택에 따라 하위 enum 항목이 바뀌는 계층적 드롭다운박스
    • 가상컬럼 활용하여 데크 뷰에 여러 컬럼 보이기, 경과시간 계산하기

    컬럼의 속성

    • Regenerate structure: 앱생성 후 컬럼이 추가/삭제/변경 되었을 때
    • Initial value: 다른 항목에서 선택한 값에 따라 특정 항목의 기본 셋팅값을 다르게 IF(), IFS()
    • App formula: 드롭다운 항목을 선택하면 다른 테이블의 이미지가 참조되도록
    • Spreadsheet formula: 구글스프레드시트의 수식을 앱시트에서 연동
    • Spreadsheet formula 응용: 다른 테이블의 입출고정보를 제품정보 테이블로 가져오기
    • Valid if: Enumlist 목록에서 다른 테이블의 값이 표시되게
    • Scannerble: 바코드를 이용하여 제품정보 입력하고 검색하기
    • Editable: 데이터를 추가/변경 상황에 따라 컬럼 편집 가능여부 다르게

    테이블 다루기

    • 키(key) 설정에 대한 여러가지 고찰
    • UNIQUEID(): 데이터 열마다 고유한 번호(key) 부여하기
    • 테이블에서 Key가 될 수 있는 컬럼이 있는지 확인하고 없으면 만들어 쓰자
    • Ref: 다른 데이터베이스의 값 참조하기
    • Ref를 적극 활용하되 하나의 관계는 한번만 설정하자
    • Is part of?: 상하위 테이블 관계 적용하기
    • 상하위 테이블 관계 응용 (부모 자식 테이블에 데이터 한꺼번에 입력)
    • Slice: 특정 조건의 데이터만 필터하여 부분 테이블로

    화면 구성

    • Sort by, Group by: 데이터를 카테고리로 묶어서 보여주기
    • Group aggregate 활용하여 집계화면 만들기
    • Format rule: 조건에 따라 항목별 서식을 다르게 하기
    • Chart view: 가상컬럼 기능을 이용하여 월별 연도별 차트 만들기
    • Dashboard view: 한 페이지에 여러개의 화면을 탭으로 넣기
    • Interactive dashboard: 선택한 항목에 따라 달라지는 반응형 차트
    • Inline view: 다른 테이블의 참조내역을 보여주는 부분 편집하기
    • Quick edit 기능으로 데이터 조회와 수정을 동시에
    • Enable QuickEdit 기능으로 여러 항목을 동시에 수정하기
    • About 화면: 앱 사용법 안내 페이지 만들기
    • Dispaly name: 동적으로 바뀌는 view name 설정하기

    액션 Action

    • 버튼을 누르면 테이블에 특정한 값을 지정해주는 액션
    • 보고 있는 데이터를 문자 메시지로 생성하는 액션
    • 다른 테이블에 데이터를 추가하는 액션 - 수불내역 관리 앱 예제
    • 보고 있는 데이터를 다른 화면의 데이터로 셋팅해주는 액션 LINKTOFORM())
    • 여러 컬럼의 값을 동시에 검색하는 액션 LINKTOFILTERDVIEW()
    • 바코드를 이용하여 여러 테이블을 동시에 검색한 후 결과를 보여주는 액션
    • 특정 테이블 전체 내용(레코드)을 삭제하는 액션
    • Enumlist에서 선택된 항목을 개별 행으로 추가하는 다중 액션
    • 반복 동작을 하는 액션, 루프 액션
    • 행 전체 선택 액션
    • 액션을 화면에 표시하거나 숨기는 여러가지 방법

    자동화 Automation Bot

    • 봇 원리와 구성, 봇 만드는 방법: Bot = Event + Process
    • 삭제한 데이터를 다른 테이블에 백업하는 봇
    • 변경된 컬럼이 무엇인지 알려주는 알림메세지 봇
    • 일정한 주기마다 데이터를 파일로 자동 저장하는 봇
    • PDF 파일을 자동으로 생성해주는 템플릿 만드는 방법

    다양한 함수 활용

    • MAXROW(): 특정 기준의 마지막행만 가져오는 슬라이스 만들기
    • SELECT(), FILTER(): 특정 조건이 일치하는 다른 행의 값을 참조하기
    • LINKTOVIEW(): Gallery view 응용하여 각 화면으로 연결되는 메뉴페이지 만들기
    • CONTEXT(): 버튼이름, 컬럼이름을 화면에 따라 다르게 표시하기
    • WEEKDAY(): 날짜 컬럼에 요일도 같이 표시 하기

    권한 설정

    • USEREMAIL(): 컬럼/테이블/앱 권한을 사용자별로 설정
    • Security filter: 각 사용자가 본인이 등록한 내용만 보이게 하기
    • User settings: 퍼블릭 앱에서 사용자 권한 설정하기
    • User settings: 퍼블릭 앱에서 입장코드로 접근 제한하기
    • User settings + Security filter: 퍼블릭 앱에서 사용자별 데이터 제한

  • 📁앱시트 사용 관련 팁

    • 내가 만든 앱의 사용자 수와 접속건수 확인 방법
    • 해결되지 않는 문제는 커뮤니티 포럼에서 알아보기
    • 동기화 설정으로 로딩속도 개선하기
    • 주소나 위치를 쉽게 입력하는 방법
    • 다른 사람이 만든 앱 소스를 공유 받고 복제하는 방법
    • 샘플 앱 만들어 공유하는 방법
    • 다른 사람의 샘플 앱을 확인하고 복사하는 방법
