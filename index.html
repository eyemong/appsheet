구글 앱시트 Google AppSheet 소개 및 사용법

by 코딩하는 약사

last updated @Mar 22, 2021

구글 앱시트 Google AppSheet는 엑셀이나 구글 스프레드시트 등을 이용하여 코딩 없이 손쉽게 데이터 기반의 모바일앱/모바일웹을 만들 수 있는 도구입니다.

이 페이지는 제가 앱시트를 직접 사용해보고 사용법을 쉽게 알아볼 수 있도록 직접 정리한 글모음입니다. 제목의 화살표를 누르면 상세 페이지 제목을 볼 수 있습니다.

  • 📁앱시트로 앱만들기 과정 간단 소개
    1. 데이터 연동하고 설정하기
    1. 기능별 화면구성하기
    1. 앱의 모양 꾸미기
    1. 규칙에 따른 액션 넣기
    1. 스마트폰에서실행하기

  • 📁앱시트 기능별 사용법
    • 데이터 열마다 고유한 번호(key) 부여하기 : UNIQUEID 함수
    • 미리 설정된 값을 선택할 수 있게 하는 데이터 속성 : Enum, Enumlist
    • 다른 데이터베이스의 값 참조하기 : Ref
    • 데이터를 카테고리로 묶어서 보여주기 : Sort by, Group by
    • 특정 조건의 데이터만 보여주기 : Slice
    • 버튼을 누르면 테이블에 특정한 값을 지정해주는 기능 : Action (set the values)
    • 보고 있는 데이터를 다른 화면의 데이터로 셋팅해주는 기능 : Action (LINKTOFORM)

  • 📁앱시트 사용 관련 팁
    • 앱시트 간략 소개 (+스마트메이커와 비교)
    • 앱시트 이용요금제 소개
    • 해결되지 않는 문제는 커뮤니티 포럼을 이용

  • 📁앱시트 이용 예제 (직접 만든 앱)
    • 전국 명절 무료주차장 조회 앱
    • 나만의 자산관리 앱 만들기 1편
    • 나만의 자산관리 앱 만들기 2편
    • 바코드 스캔 앱 만들기
