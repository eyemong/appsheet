구글 앱시트 Google AppSheet는 엑셀이나 구글 스프레드시트 등을 이용하여 코딩 없이 손쉽게 데이터 기반의 모바일앱/모바일웹을 만들 수 있는 도구입니다. ▷ 노코드 앱제작 툴 앱시트 간략 소개

이 페이지는 제가 앱시트를 직접 사용해본 후 국내 사용자들이 쉽게 배워보실 수 있도록 정리한 글모음입니다. 아래 각 폴더의 제목을 누르면 상세 페이지 링크를 볼 수 있습니다. 키워드로 관련 글을 찾고자 하는 경우 Ctrl + F를 눌러 검색해보시기 바랍니다.

  • 앱시트 기능별 앱 제작방법

    테이블 구조

    • 행마다 고유한 번호(Key) 부여하기 - UNIQUEID 함수
    • 어느 컬럼을 Key로 선택할까? 연속적으로 증가하는 Key는 어떻게 만드나?
    • 테이블에서 키(Key)가 될 수 있는 컬럼이 있는지 확인하고 없으면 만들어 쓰자.
    • 다른 데이터베이스(테이블)을 연결하고 참조하는 Ref 컬럼 타입
    • Ref 타입을 이용해 다른 테이블에 연결된 이미지를 참조하여 불러오기
    • Ref 타입을 적극 활용하되 하나의 관계는 한번만 설정하자.
    • 상하위 테이블 관계 설정 - Is a part of?
    • 상위 테이블에서 하위 테이블의 컬럼 참조하기
    • 하위 테이블에서 상위 테이블 컬럼 참조하여 한 번에 여러 행 입력하기
    • Ref 설정을 하면서 드롭다운 목록은 안 보이게 하기
    • Ref 드롭다운 생성 시 원하는 조건에 맞는 목록만 보이게 하기 - Valid if
    • 특정 조건의 데이터만 필터링해서 보여주는 슬라이스(Slice) 기능
    • 특정 조건의 마지막 행만 가져오는 슬라이스 만들기 - MAXROW 함수
    • 최근 한 달 데이터만 볼 수 있는 슬라이스 만들고 차트 뷰로 연결하기 📺
    • 구글 캘린더(Google Calendar) 연동하기

    컬럼의 유형과 설정

    • 미리 설정된 목록 중 선택할 수 있게 하는 컬럼 타입 Enum, EnumList
    • 상위 Enum 선택에 따라 하위 Enum 항목이 바뀌는 계층적 선택지 만들기
    • EnumList (다중 선택) 목록에서 다른 테이블의 값이 표시되게 하기 - Valid if
    • 폼 뷰에서 입력할 값 추천 받기 - Suggested values
    • 이미지 삽입, 이미지 연동 방법 - Image 타입
    • 구글 스프레드시트에서 앱시트 업로드 이미지를 바로 미리보기하는 방법
    • 구글 드라이브에서 공유한 이미지나 파일을 앱시트에서 링크하는 방법
    • Url 타입 이용하여 검색사이트 연결하기 - ENCODEURL 함수
    • Url 타입 이용하여 지도앱, 네비앱 (네이버지도, 카카오맵, 티맵) 연결
    • 주소 자동 완성(Address 타입), GPS 현재 위치 입력(LatLong 타입)
    • 서명 넣기(Signature 타입)와 그림 그려서 넣기(Drawing 타입)
    • 컬럼 내용을 수정한 일시와 수정한 횟수를 캡쳐하는 ChangeTimestamp, ChangeCounter 타입
    • LongText 타입에서 마크다운(Markdown) 문법이나 HTML 태그로 서식 적용하기
    • 바코드나 QR코드를 이용하여 제품정보 입력하고 검색하는 Scannable 기능
    • SEARCH? 속성을 활용한 검색 필터 기능
    • 데이터 추가/변경 상황에 따라 편집 가능 여부 다르게 설정하기 - Editable?, IN 함수
    • Enum 값에 따라 컬럼이 보였다 숨겼다 하기, 값이 없어도 디테일 뷰에 표시되게 하기 - SHOW?

    화면 구성

    • 데이터를 계층적 그룹으로 묶어서 보여주기 - Group by
    • Group aggregate 활용하여 집계화면 만들기
    • 구글 스프레드시트의 QUERY 기능을 이용하여 집계화면 만들기
    • 새로고침 할 때마다 랜덤 정렬하기 - Sort by, RANDBETWEEN 함수
    • 조건에 따라 다른 서식 적용하기 - Format Rules
    • Group by에 의한 그룹명에 포맷 룰(Format Rule) 서식 적용하기
    • 한 페이지에 여러개의 화면을 탭으로 넣기 - Dashboard Mobile Tab view
    • 선택한 항목에 따라 달라지는 반응형 차트 - Interactive Dashboard
    • 폼 뷰에서 안내 문구나 이미지 등 보여주기 목적의 컬럼 만들기 - Show 타입
    • Show 타입으로 여러 페이지나 여러 섹션으로 된 폼 뷰 만들기
    • 주소 또는 위도 경도 데이터로 맵 뷰(Map view) 만들기
    • XY type을 이용하여 지도 대신 커스텀 배경의 맵 뷰(Map view) 만들기 📺
    • 동적으로 바뀌는 뷰 네임(View name) - Display name
    • 버튼이름, 컬럼이름을 화면에 따라 다르게 표시하기 - CONTEXT 함수
    • 다른 테이블의 참조내역을 보여주는 인라인 뷰(inline view) 편집하기
    • 디테일 뷰에서 데이터 조회와 수정을 동시에 - Quick edit 기능
    • 목록 뷰에서 여러 컬럼을 동시에 수정 - Enable QuickEdit 기능
    • 앱 사용법 안내 페이지 만들기 - About 화면
    • 갤러리 뷰를 이용해 각 화면으로 연결되는 메뉴 페이지 만들기 - LINKTOVIEW 함수
    • 뷰에서 보이는 컬럼 순서 바꾸기 - Column order
    • 메뉴바의 메뉴 순서 바꾸기 - Position
    • 캘린더 뷰 만들기

    여러가지 함수 활용

    • 가상컬럼 만드는 법과 연도별 월별 차트뷰 만들기 - YEAR, MONTH 함수
    • 주단위 집계, 월단위 집계 화면 만들기 - WEEKNUM 함수
    • 날짜 컬럼에서 요일도 표시 하기 - WEEKDAY 함수
    • 데크 뷰에 여러 컬럼값 보이게 하기, 경과시간 계산하기
    • 경과년도, 경과개월, 경과년월 구하기 - 구글 스프레드시트 DATEIF 함수와 비교
    • 다른 컬럼에서 선택한 값에 따라 컬럼의 기본값을 다르게 셋팅하기 - IF, IFS 함수
    • 특정 조건이 일치하는 다른 행의 값을 참조하는 방법 - SELECT, FILTER 함수
    • 기존의 행 데이터를 불러오는 자동완성 기능 만들기 - SELECT 함수
    • 로그인 계정을 인식하여 이름으로 바꾸기 - LOOKUP 함수
    • 중복값 입력 방지 - Key 또는 Valid if
    • 연도별 분류별 연속적 숫자(시리얼 넘버) 부여하기
    • 구글 스프레드시트 함수 연동하기 - Spreadsheet formula 📺
    • 구글 스프레드시트 함수 연동하기 - Spreadsheet formula 📝
    • 구글 시트 SUMIF 함수로 입출고정보를 제품정보 테이블에 적용
    • 앱시트에서 유용하게 쓰이는 함수들의 문법과 활용사례 요약 정리
    • 수학 계산 관련 함수 종류, 문법과 응용

    동작(액션, Action) 만들기

    • 지금 보고 있는 데이터를 문자 메시지로 생성하는 액션 버튼 만들기
    • 액션(Action)으로 값을 바꾸는 버튼 만들기 📺
    • 특정 컬럼에 값을 셋팅하는 액션 버튼 만들기 📝
    • 지금 보고있는 데이터를 폼 뷰에 셋팅해주는 액션 - LINKTOFORM 함수
    • 액션(Action)을 화면에 표시하거나 숨기는 여러가지 방법
    • 기존의 행을 복사하는 액션
    • 특정 테이블의 전체 행(레코드)을 삭제하는 액션
    • 행 전체를 한꺼번에 선택하는 액션 버튼 만들기
    • 맵 뷰(지도)에서 현위치/특정 위치로 이동하는 액션
    • 반복 동작을 하는 액션, 루프 액션, 액션으로 반복문 만들기
    • Enumlist에서 선택된 항목을 개별 행으로 추가하는 다중 액션
    • 특정 조건에 맞는 결과만 표시하는 액션 - LINKTOFILTEREDVIEW 함수, CONTAINS 함수
    • 여러 테이블을 동시에 검색한 후 결과를 보여주는 액션
    • 푸쉬 알림 봇을 클릭하면 앱시트에서 원하는 화면으로 이동하기
